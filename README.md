# 🚀 spcinv Game 🎮

Welcome to your own spcinv game! 🌌

## 🎯 Game Features

- 🛸 Player ship with smooth movement controls
- 🔫 Laser blaster mechanics with spacebar
- 👾 Enemy waves with different colors and behaviors
- 💥 Particle explosion effects
- 📊 Score tracking and lives system
- 🌟 Starry space background

## 🕹️ How to Play

- ← → Arrow Keys: Move your ship left and right
- SPACEBAR: Fire your laser
- Destroy all enemies to advance to the next level
- Avoid enemy lasers to preserve your lives

- `index.html`: Main game structure and UI
